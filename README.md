# 🎮 PUBG Mobile Sensitivity Bot

بوت تليجرام متكامل لإعدادات حساسية ببجي موبايل

## ✨ المميزات

- 🎯 إعدادات حساسية النار (مع/بدون جيروسكوب)
- 🔫 إعدادات خاصة للقناصات (AWM, M24, Kar98k)
- 🖐️ أساليب اللعب (1/2/3/4 أصابع و Claw)
- 🏆 نظام مستويات وخبرة وإنجازات
- 📊 تحليل أداء اللاعب (نسبة الفوز، K/D ratio)
- 🎨 أزرار تفاعلية للتنقل السهل
- 📤 تصدير/استيراد الإعدادات كـ JSON
- 🌙 وضع الليل/النهار

## 🚀 الأوامر الرئيسية

| الأمر | الوظيفة |
|-------|---------|
| `/start` | بدء البوت |
| `/sensitivity_full` | جميع إعدادات الحساسية |
| `/sniper` | إعدادات القناصات |
| `/profile` | ملفك الشخصي ومستواك |
| `/performance` | تحليل أدائك |
| `/export` | تصدير إعداداتك |

## 📦 النشر على Render

1. ارفع الكود إلى GitHub
2. أنشئ Web Service جديد على Render
3. الصق رابط المستودع
4. أضف المتغيرات البيئية (TOKEN, ADMIN_ID)
5. انتظر حتى يتم النشر
